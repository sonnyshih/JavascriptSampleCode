# 一、執行 TypeScript
  1. 安裝 TypeScript     
        > $ npm install -g typescript
  
  2. compile ts 檔案 (會產生HelloWorld.js檔案)
        > $ tsc HelloWorld.ts   
  
  3. 執行js檔案
        > $ node HelloWorld.js
